# Keen Keeper

Keen Keeper is a relationship tracking web app that helps you manage meaningful connections, monitor communication status, and keep your follow-ups on time.

## Technologies Used

- React 19 + vite
- React Router
- Tailwind CSS + DaisyUI
- Recharts
- React Toastify
- Lucide React

## Key Features

1. Smart contact overview:

- Track friends with status labels like On-track, Almost Due, and Overdue using an easy-to-scan dashboard.

2. Detailed friend profile actions:

- Open a friend profile and quickly log Call, Message, or Video interactions with instant toast feedback.

3. Timeline and interaction insights:

- View interaction history, filter timeline events by type, and monitor monthly interaction totals.
